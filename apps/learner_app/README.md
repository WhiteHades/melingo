# Melangua Learner App

Flutter client for Melangua, an offline-first speaking-first language tutor.

## What lives here

- Adaptive Flutter shell for phone, tablet, desktop, and web.
- Local-first speaking flow with ASR, tutoring, and TTS hooks.
- Firebase bootstrap and generated platform configuration.
- Encrypted local persistence for sensitive learner state.

## Core commands

```bash
flutter pub get
flutter analyze
flutter test
```

## Firebase notes

- `lib/firebase_options.dart` is generated by FlutterFire.
- Use `USE_FIREBASE_EMULATORS=true` to point the app at the local Emulator Suite.
- App Check and anonymous auth bootstrap are scaffolded behind environment flags so local development stays unblocked.
