# Melangua Learner App

Flutter client for Melangua, an offline-first speaking-first language tutor.

## What lives here

- Adaptive Flutter shell for phone, tablet, desktop, and web.
- Local-first speaking flow with ASR, tutoring, and TTS hooks.
- Firebase bootstrap and generated platform configuration.
- Encrypted local persistence for sensitive learner state.

## Core commands

```bash
flutter pub get
flutter analyze
flutter test
```

Additional platform checks:

```bash
flutter build web --no-wasm-dry-run
flutter build linux
flutter build apk --debug
```

## Current implementation note

- The app shell, onboarding, review, stats, local persistence, Firebase bootstrap, and sync scaffolding are implemented.
- The current ASR, tutor, and TTS path uses the simulated `UnimplementedAiBridgePlatform` during local runs and tests.
- Manual product-flow testing is still useful, but it does not yet validate real on-device voice model quality.

## Firebase notes

- `lib/firebase_options.dart` is generated by FlutterFire.
- Use `USE_FIREBASE_EMULATORS=true` to point the app at the local Emulator Suite.
- App Check and anonymous auth bootstrap are scaffolded behind environment flags so local development stays unblocked.

See `../../docs/manual-testing.md` for a full runbook covering Linux, web, Android, backend, and smoke-test scenarios.
